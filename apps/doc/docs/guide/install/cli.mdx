# Using the CLI

## 1. Login with wrangler on your terminal
```shell
npx wrangler login
```

## 2. Run the npflared installer
```shell
npx @npflared/cli install
```

And we are done 🎉

:::tip
For more information about the CLI and all available commands, see the [CLI Reference](../cli.mdx).
:::
